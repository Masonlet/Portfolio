## Building the Project
### 1. Clone the Repository
```bash
# Clone Portfolio
git clone https://github.com/masonlet/portfolio.git
cd portfolio

# Install and start
npm install
```

Create `.env` file from `.env.example` and add your Resend API key.

```bash
# Start Server
npm start
```
